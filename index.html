Project Management System

Prototype of a customizable project management system with third-party integration of Slack and GitHub, made for Integrify Winter Hackathon 2023:

  • Posting messages to Slack with incoming webhooks when new tasks are created.
  • Connected to a NoSQL MongoDB database.
  • Fetching data from GitHub API to turn Issues from a repository into new tasks on a kanban board.
  • Pitched and presented the winning project to a panel of judges.
TypeScript React Redux Material UI Node Express MongoDB Webhooks Figma

Library Management App

Library management app to browse, borrow and return digital books:

  • User, admin and book management system running on a Node server with Express.
  • Connected to a NoSQL MongoDB database.
  • Token based authentication with cookies and JWT.
  • Scalable, reusable and responsive frontend made with React and MUI.
  • Handles form data and validation with Formik, and CRUD operations over HTTP.
